Distribution Void Analysis at Scale

$120–133Mannualized opportunity surfaced
3.8B+rows queried, store-UPC-week grain
~1,500region/product combinations flagged
PROBLEM APPROACH STACK IMPACT

Distribution gaps across thousands of store/product combinations were invisible at the granularity leadership needed to act on them.

Built a multi-phase Databricks SQL analysis against a 3.8B+ row Snowflake-federated fact table, then engineered a custom pre-aggregation layer (FactVoidPeriod) feeding a Power BI star schema — period-over-period matrices with DAX measures for open voids, net closed voids, and % close rates.

Databricks SQL, Snowflake federation, Power BI star schema, DAX, heatmap conditional formatting.

Surfaced ~$120–133M in annualized distribution opportunity. Became a standing VP-facing report driving close-rate accountability.

API-Driven Data Validation: Retail Shelf Capacity

PROBLEM APPROACH STACK IMPACT

Planogram capacity files appeared to conflict heavily with a retailer’s out-of-stock API data — a discrepancy headed straight for leadership.

Built a comparison engine between planogram CSVs and the retailer’s API, then root-caused the gap instead of reporting it at face value.
